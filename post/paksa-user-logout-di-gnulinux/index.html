Paksa user untuk logout (GNU/Linux)

Tuesday , 13 Jan 2015 06:20:00 linux command-line

Untuk memaksa user yang sedang logout di mesin Unix atau Gnu/Linux bisa menggunakan perintah pkill, berikut contohnya:

# pkill -KILL -u username Perintah diatas akan memaksa user dengan login name username untuk logout.

Jika hanya ingin melakukan stop sementara opsi -KILL bisa diganti dengan -STOP dan -CONT untuk melanjutkan kembali.
