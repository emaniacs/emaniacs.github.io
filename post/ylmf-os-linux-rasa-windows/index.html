ylmf Os, Linux Rasa Windows

Wednesday , 06 Apr 2011 21:21:00 gnu linux

Ylmf Os, merupakan sebuah Linux yang mirip secara tampilan dengan Windows. bahkan persis banget, meski saya sendiri tidak menyukai tampilan windows yang kurang fleksibel tapi os ini sangat bagus untuk pengenalan Linux kepada user baru yang telah terbiasa dengan Windows (mungkin kurang tepat kalau disebut menipu), sistem operasi berbasis kernel Linux 2.6.32-22.33 ini menggunakan xorg 1.7.6 dan sebagai desktop menggunakan gnome 2.30.0.

Sistem operasi ini dikembangkan oleh orang China yang bernama Lailinfeng, yang berbasiskan Ubuntu 10, versi terbaru dari sistem ini adalah versi 3.0 http://www.ylmf.org/en/images/snaps/YlmfOS-8.jpg

Berikut beberapa fitur dari ylmf Os yang saya kutip dari situs resminya
System

  • wine : emulator program windows untuk linux.
  • ccsm : sebagai pengatur settingan Compiz
  • ailurus : aplikasi penginstalan, agar penginstalan aplikasi Linux jadi lebih mudah.
  • p7zip : aplikasi untuk pengelolaan file-file kompresi di Linux.
  • ndisgtk, samba(sharing folder), leafpad(editor).

  •  gmount-iso(untuk mounting file iso, atau virtual cd). Internet

  • firefox sebagai browser default.

  • Emphaty : chat client untuk linux.

  • aMule : aplikasi untuk sharing file secara peer to peer.

  • Iptux : chatting berdasarkan Ip untuk linux.

  • gFtp : ftp klient.

  • Xchat : irc client (aplikasi untuk chating di irc server.

  • Emesene : Instant Messenger untuk Windows Live Messenger.

  • Transmision : bitTorrent client.

  • vinagre : VNC Client. Multimedia

  • SMPlayer : front-end dari aplikasi MPlayer sebagai pemutar video

  • Audacious : pemutar musik.

  • mid3iconv : untuk konversi tag ID3 menjadi ID3v2 format.

  • PiTiVi : Video editor untuk GNOME Desktop.

  • gThumb : aplikasi untuk melihat gambar.

  • cheese : aplikasi untuk mengambil gambar atau video dari webcam. Aplikasi Kantoran.

  • OpenOffice : pengolah kata, presentasi dan aplikasi untuk kantor lainnya.

  • chmsee : melihat file berformat chm di Linux.

  • Evince : sebagai pdf reader dan juga mendukung format lainnya.

  • Gnote : catatan di desktop.

  • Simple Scan : aplikasi untuk melakukan scanning dokumen.

  • Stardict : aplikasi kamus Lain-lain

  • Bluefish : editor untuk programmer dan webdesigner.

  • Geany : editor  GTK+ yang cukup ringan berbasiskan Scintilla.

  • Vim : text editor yang powerfull Untuk lebih jelasnya bisa dilihat langsung di situs resminya, http://www.ylmf.org
