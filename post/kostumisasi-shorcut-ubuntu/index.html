Kostumisasi Shorcut Ubuntu

Sunday , 25 Apr 2010 04:53:00 gnu linux

Penggunaan keyboard dalam hal sebagai shortcuts sangat penting apalagi jika kita ingin mempercepat untuk memanggil suatu aplikasi.

Dengan Linux dalam hal ini Ubuntu kita dengan sangat mudah membuat shortcuts untuk sebuah aplikasi yang ingin kita jalankan, bahkan untuk komputer yang berlambang jendela di keyboard pun bisa kita gunakan untuk sebuah shortcuts.

Salah satu caranya bisa dengan mengikuti trik berikut:

  1. Klik pada Main Menu → System → Preferences → Keyboard

  2. Kemudian pada tab Layout klik Layout Option

  3. Kemudian klik pada pilihan Alt/Win key behavior

  4. Dan aktifkan pilihan Super is mapped to Win keys

Setelah ini maka keyboard anda khususnya tombol WinKeys telah dikenali.

Untuk membuat shortcuts bisa dengan Keyboard shortcuts pada pilihan Main Menu → System → Preferences → Keyboard Shortcuts.

Disini anda bisa menambahkan aplikasi apa yang ingin anda panggil dengan sebuah shortcutss, pada saat menambahkan shortcuts dengan menggunakan kombinasi tombol winkey akan tertulis sebagai , dan semua ini akan dicatat pada gconf-editor pada key /apps/metacity/global_keybindings.
