Perintah Linux 1

Sunday , 25 Apr 2010 04:52:00 gnu linux


Berikut ini adalah beberapa perintah perintah Linux sederhana namun sangat powerfull. Perintah ini saya dapat dari kegiatan oprek Linux yang dilakukan sendiri, ada juga dari internet, dari tutorial posting dari para pencinta Linux yang saya baca dari Internet.

  • ifconfig -a
    Perintah ini melihat konfigurasi network pada lan card, lan card sendiri biasanya dinamai dengan eth0, eth1 dan seterusnya. Jika ingin melihat konfigurasi ‘hanya’ pada satu device network bisa digunakan “ifconfig “.

  • #ifconfig eth0 192.168.1.15⁄24 up
    Perintah ini untuk memberikan ip address dan netmasknya (24) pada device eth0 sekaligus untuk mengaktifkannya.
    Untuk lebih lanjut bisa dilihat “ifconfig –help” atau “man ifconfig”.

  • #route add default gw 192.168.1.1
    Menjadikan host 192.168.1.1 sebagai default gateway.

  • iwlist wlan0 scan
    Melakukan scanning access point wireless.
