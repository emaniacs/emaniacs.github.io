Emacs Command line – (Standard Options)

Sunday , 13 Mar 2011 01:13:00 gnu linux gnu emacs

Beberapa opsi standar Emacs :

  • namaFile
    **$emacs "namafile"**
    artinya kurang lebih membuka file “namafile” dengan emacs.

  • –file namaFile, –find-file namaFile, –visit namaFile
    Perintah kurang lebih sama dengan perintah pertama diatas.

  • +number
    Ex: **$emacs +12 "namafile"**
    Membuka file “namafile” dan loncat ke baris 12, kolom pertama.

  • +line:column
    Sama dengan yang diatas tapi ditambahkan dengan opsi kolom yang ingin dituju

  • -q, –no-init-file
    Menjalankan Emacs dengan tidak meload file init atau file konfigurasi Emacs. Secara default file init yang akan diload emacs yaitu file ~/.emacsatau ~/.emacs.d/init.el.
    File ini merupakan file untuk melakukan settingan pada emacs misalkan untuk melakukan konfigurasi shortcuts keyboard ketika pengeditan atau hal-hal lainnya.
    Jika dijalankan dengan opsi ini maka konfigurasi emacs akan sama ketika pertama kali emacs dijalankan, atau default.

  • –no-site-file

  • –no-desktop

  • -nl, –no-shared-memory

  • -Q, –quick
    Sama dengan –no-init-file namun ditambahkan dengan tidak meload splash screen Emacs atau opsi ini sama dengan –no-init-file –no-splash.

  • –no-splash
    tidak meload splash screen emacs ketika dijalankan.

  • –debug-init
    Melakukan debug atau mencari kesalahan pada file init (~/.emacs atau~/.emacs.d/init.el) ketika menjalankan emacs, hal ini berguna ketika kita melakukan konfigurasi emacs dengan file init dan mencari kesalahan sintaks pada file tersebut.

  • -t namaFile, –terminal namaFile

  • –multibyte, –no-unibyte

  • –unibyte, –no-multibyte

  • –version
    Menampilkan versi dari emacs dan keluar aplikasi.

  • –help
    Menampilkan help/bantuan dari emacs dan keluar aplikasi.
