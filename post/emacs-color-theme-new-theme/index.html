Emacs Color Theme ( new Theme )

Monday , 11 Apr 2011 23:09:00 gnu emacs

Setelah lelah membaca buku, capek mengetik kode ane bingung mau ngapain, trus coba jalan-jalan http://www.emacswiki.org/ trus kesasar di bagian Color Theme, ternyata banyak banget tema untuk Emacs.

Akhirnya mutusin untuk coba ganti tema Emacs, dengan nuansa gelap, karena sebelumnya warna tema Emacs ane, berwarna cerah.

Secara default jika download color-theme pun telah tersedia berbagai tema, yang bisa langsung dipakai.

Cara Instalasai Tema.

  1. Download color-theme.el disitus resminya http://www.nongnu.org/color-theme/ atau bisa langsung download disini, http://download.savannah.gnu.org/releases/color-theme/.
    ** $ wget “http://download.savannah.gnu.org/releases/color-theme/color- theme-6.6.0.tar.gz”**

  2. Pilih versi paling tinggi (saat ini versi 6.6.0), kemudian extract file tersebut,
    ** $tar xf color-theme-6.6.0.tar.gz**

  3. Install dengan perintah make ; make install.
    _ $ cd color-theme-6.6.0
    $ make
    $ sudo make install_

  4. Kemudian edit file .emacs di direktori home, tambahkan kode berikut
    ** (require ‘color-theme-autoload “color-theme-autoloads”)**

  5. Jalankan Emacs, untuk mengganti theme tekan tombol Alt+x color-theme Tab dan tekan Tab untuk melihat popup color-theme yang tersedia.

Emacs dengan tema standar

Setelah ane coba semua, terasa belum puas akhirnya mutusin untuk bikin tema sendiri, walau cukup ribet karena harus tahu kode-kode warna untuk html, tapi kalo di emacs bisa dengan menekan tombol Alt+x list-color-display Ret untuk melihat daftar warna.

Setelah berhasil membuat sebuat tema, tema ini diberi nama “smooth-in-the- dark” seperti namanya tema ini cukup nyaman dilihat meski backgroundnya gelap, berikut kodenya,

(defun color-theme-smooth-in-the-dark ()  
  "created by Ardi   
  (interactive)  
  (color-theme-install  
   '(color-theme-smooth-in-the-dark  
      ((background-color . "#393b3c")  
      (background-mode . light)  
      (border-color . "#544f4f")  
      (cursor-color . "#0f576c")  
      (foreground-color . "#b7b7a9")  
      (mouse-color . "black"))  
     (fringe ((t (:background "#544f4f"))))  
     (mode-line ((t (:foreground "#eeeeec" :background "#666765"))))  
     (region ((t (:background "#2b4855"))))  
     (font-lock-builtin-face ((t (:foreground "#1fc4e5"))))  
     (font-lock-comment-face ((t (:foreground "#bc7677"))))  
     (font-lock-function-name-face ((t (:foreground "#ffee61"))))  
     (font-lock-keyword-face ((t (:foreground "#4f8ac9"))))  
     (font-lock-string-face ((t (:foreground "#c68d5d"))))  
     (font-lock-type-face ((t (:foreground"#6fe000"))))  
     (font-lock-variable-name-face ((t (:foreground "#f80dd0"))))  
     (minibuffer-prompt ((t (:foreground "#03aa07" :bold t))))  
     (font-lock-warning-face ((t (:foreground "Red" :bold t))))  
     )))  
(provide 'color-theme-smooth-in-the-dark)

Simpan kode diatas dengan nama color-theme-smooth-in-the-dark.el simpan didirektori konfigurasi Emacs, kemudian tambahkan kode berikut di file .emacs.
(color-theme-smooth-in-the-dark)
Perintah ini harus dipanggil setelah perintah (require ‘color-theme) seperti yang tertera diatas.

Emacs dengan tema smooth-in-the-dark

Menurut ane, warnanya cukup nyaman, silahkan nilai masing2.

Kalo capek dan ga mau repot dengan kode warna-warna untuk tema diatas bisa dengan menggunakan color-theme generatornya disini  , ane pun awalnya menggunakan generator ini.
